# Project Audify

## Dependencies

- [GNU make](https://gnu.org/software/make)
- [Cmake](https://cmake.org)
- [Rust](https://rust-lang.org)
- [NodeJs v24 or greater](https://nodejs.org)
- [Clang/g++](https://)
- [just](https://just.systems)

## Getting started

```sh
git clone https://github.com/opeolluwa/audify
cd audify
just configure
```
